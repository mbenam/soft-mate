About This Tool

This tool is a Semantic UI Layout Generator. AI Vision models struggle to deduce perfect math, alignments, and programmatic groupings from pixel screenshots. By drawing the UI here, you bypass the vision model completely. You produce a mathematically perfect, logically grouped JSON schema that tells the AI exactly where elements live, how they are colored, and how they behave when selected.

How To Operate

  • Draw: Select a logical color from the palette, click any cell on the grid, and type. For sliders, select the SLIDER palette and press any key to draw solid blocks. Spacebar clears a cell.
  • Select (Copy/Paste/Move): Drag a box to highlight elements.
    • Move: Click and drag inside the orange box to move the elements.
    • Duplicate: Hold Alt while dragging the box to leave a copy behind.
    • Copy/Paste: Press Ctrl+C to copy the highlighted area. Click anywhere on the grid (or draw a new 1x1 selection box) and press Ctrl+V to paste.
    • Nudge: Press the Arrow Keys to precisely bump the selected group around.
    • Delete: Press Delete or Backspace to clear the selected area.
  • Tag Field (Interactive): M8 fields are grouped (e.g., Label + Hex Value + Slider). Use this tool to drag a box over related elements and assign them a shared ID (like CUTOFF).
  • Tag Var (Dynamic Text): Use this tool to tag text that isn't selectable but changes programmatically (e.g., drawing INST. 00 and tagging it as SCREEN_TITLE).
